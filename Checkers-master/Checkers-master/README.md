# Checkers
Simple Checkers game in Unity.


## Built with

* [Microsoft Visual Studio 2017](https://visualstudio.microsoft.com/pl/downloads/) - IDE
* [Sourcetree](https://www.sourcetreeapp.com/) - version control
* [Unity3D](https://unity3d.com/) - game engine
